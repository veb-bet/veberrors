# veberrors
Маленький Flask-сервер с прикольными кастомными HTTP-ошибками для добавления юмора в ваше API!

## Доступные ошибки

### 430 Too Many Dad Jokes
Слишком много папиных шуток. Сервер возвращает случайную шутку из коллекции 100+ папиных шуток!

### 472 Cat Interruption  
Кот прошёл по клавиатуре. Прерывание запроса пушистым помощником.

### 469 Overly Sarcastic Request
Клиент был слишком саркастичен. Сервер не смог ответить серьёзно.

### 418 I'm a Teapot (расширенная)
Сервер занят завариванием чая и не может обработать запрос.

### 525 Coffee Brewing
Сервер варит утренний кофе. Без кофеина он не может работать.

### 580 Procrastination Error
Сервер откладывает обработку запроса на потом (занят просмотром котиков).

### Time Travel Conflict
Обнаружен временной парадокс. Ответ был отправлен в прошлом.

### No Response
Серверу лень отвечать. Попробуйте позже или предложите кофе.


## Установка и запуск

Установи зависимости:
```
pip install flask
```

Запусти сервер:
```
python funny_error.py
```

Сервер стартует на http://127.0.0.1:5000/

## Примеры запросов
1. Слишком много папиных шуток
```
curl -i -X POST http://127.0.0.1:5000/tell-a-dad-joke \
  -H "Content-Type: application/json" \
  -d '{"joke":"папа пошутил"}'
```

2. Кот прошёл по клавиатуре
```
curl -i http://127.0.0.1:5000/cat
```

3. Сарказм не прошёл
```
curl -i http://127.0.0.1:5000/sarcasm
```

4. Сервер - чайник
```bash
curl -i http://127.0.0.1:5000/tea
```

5. Сервер варит кофе
```bash
curl -i http://127.0.0.1:5000/coffee
```

6. Прокрастинация
```bash
curl -i http://127.0.0.1:5000/procrastinate
```

7. Парадокс времени
```bash
curl -i http://127.0.0.1:5000/time-travel
```

8. Ленивый сервер
```bash
curl -i http://127.0.0.1:5000/lazy
```

Это нестандартные HTTP-коды, поэтому браузеры могут отображать их как UNKNOWN.
Но Flask и curl корректно передают и принимают такие ответы.
